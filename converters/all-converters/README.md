# Data-convertion

Here you can convert annotations from one format to another .

    1. xml to csv  
    2. xml to txt  
    3. txt to xml  
    4. txt to csv  
    5. csv to xml  
    6. csv to txt   
    7. csv to json

Requirements:

       os
       csv
       glob
       json
       lxml
       numpy
       pandas
       pillow
       argparse
